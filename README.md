 LaTeX-Beamer Präsentation
===========================
Im Toolbox Theme...

 Installation:
-----------
 + Installiere dir pdflatex
 + Installiere evince als PDF Viewer 

 Präsentieren
--------------
```bash
# generate a pdf from your tex
make

# generate a pdf and open it in the viewer
make view

# Clean all up
make distclean
```
